FlyRodCrosby
============

Making Cryptoparties easier since 1897!

FlyRodCrosby[1] is an automated installer for all of the standard Cryptoparty tools.  It currently handles:

GPG4Win

We plan to support:

Thunderbird with Enigmail
Tor Browser Bundle
Cryptocat
